Dependencies:

* Python 3
* Node.js

Installations Instructions:

-> Run command:
    pip install Django djangorestframework django-cors-headers mysqlclient xhtml2pdf
-> Navigate to project backend directory and run command:
    python manage.py runserver
-> Run command:
    npm i lite-server
-> And then:
    lite-server --baseDir="path to frontend directory"
-> No login username or password is required.
-> Database is built on Amazon RDS.
